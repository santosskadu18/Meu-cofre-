# Meu Cofre — Gestão Financeira Pessoal

Um webapp standalone para controlar suas finanças pessoais.

**Funcionalidades:**
- 📊 Visão geral com gráficos
- 💰 Registro de receitas, despesas e investimentos
- 🏦 Gestão de contas e cartões de crédito
- 📈 Investimentos com projeção de rendimento por CDI
- 📋 Categorias com tetos de gastos
- 💾 Dados salvos localmente no navegador

**Como usar:**
Basta abrir `index.html` no navegador (ou acessar via link se hospedado).

**Sem dependências externas** — tudo funciona offline depois de carregar uma vez.
